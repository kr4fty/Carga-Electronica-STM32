* secuencuencia

Menu -> Modos -> Tiempo

* Se imprime reloj
    00:00:00
    ^------------- el primero con los colores invertidos (o titilando)
* moviendo el encoder se ira cambiando a colores invertidos
    HH -> MM -> SS -> HH

* Haciendo click puedo variar el selecdionado
   HH de 0 a 23
   MM de 0 a 59
   SS de 0 a 59

si HH supera el 23 se pasa a 00:00:00
si MM supera el 59, se pasa HH+1 -> +1:00:xx
si SS supera el 59, se pasa MM+1 -> xx:+1:00

* con

* Haciendo doble click, o click largo, se supone que ya ha configurado el tiempo
    se guarda HH, MM, y SS (tambien se podria pasar a tiempo total, seria mas facil de comparar luego)

* Se limpia memoria, reconfigura el encoder y se sale
